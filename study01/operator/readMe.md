#연산자
1. 산술연산자
 - 연산자 '+'는 피연산자의 자료형을 int로 면환한 후 덧샘을 수행
 - char형 변수 c2에 c2++; 대신 c2=c2+1을 하면 에러가 난다.
 - +의 연산 결과로 int형으로 형변환 하기 때문이다. char형으로 형변환을 하자.
 > 
         char c1 = 'a';
         char c2 = c1;
         char c3 = ' ' ;
 
         int i = c1+1;
         c2++;
         //c2 = c2+1;  // 애러남
         c2 = (char)(c2+1);
